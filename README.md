# CBDC-Project-Demo

## Windows上部屬

git下載
```
git clone https://github.com/AlexTrinityBlock/CBDC-Project-Demo.git
```
進入資料夾
```
cd CBDC-Project-Demo
```
裝設虛擬環境
```
py -m virtualenv BankServer 
```
進入資料夾
```
cd BankServer
```
啟動虛擬環境
```
Scripts\activate.bat
```

安裝所有需要套件
```
py -m pip install -r requirements.txt
```

測試
```
python test.py
```
